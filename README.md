# [과제]Glim_Project

## 작업 사항
1. 랜덤하게 원의 위치를 구하여 원을 그리는 루틴 구현
2. 중심좌표를 구하여 십자표시로 표출
3. 원 바깥 외각선 테두리를 노란색으로 색칠

## 작업 환경
1. Dev Tool Version : Visual Studio 2022
2. Windows SDK Version : 10.0.22000.0
